## 변수 주석

  변수: 데이터를 담아 놓기 위해 이름표를 붙여놓는 공간
  
  var 변수이름 = 변수값; 
  ```
  //변수 선언
  var num;
  //변수 초기화
  num=10;
  //변수 선언+초기화
  var num=10;
  num=20;
  
  console.log(num); //20
  ```
